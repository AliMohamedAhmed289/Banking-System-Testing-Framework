# Banking System Testing Framework

## Project Title
Comprehensive Testing Framework for Parabank Banking System

## Project Description
This project aims to design and implement a complete testing framework for the Parabank online banking system.

The project includes:
- Manual Testing (Test Plan, Test Scenarios, Test Cases in Excel)
- Automation Testing using Java, Selenium WebDriver, TestNG
- Page Object Model (POM)
- Data Driven Testing (DDT)
- Cucumber (BDD)
- API Testing using Postman & JavaScript
- Database Validation using SQL Server
